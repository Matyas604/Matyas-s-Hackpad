# Matyas-s-Hackpad
It has 7 buttons, a rotary encoder, and an OLED display. Each key is backlit with SK6812 MINI-E LEDs. It uses QMK firmware.
## CAD Model:
Everything is connected by four M3x16mm screws and M3x5mx4mm heatset inserts. The case is printed in two parts, top and bottom.
