# Firmware

**QMK**: This is a port of my hackpad to QMK firmware.
It is connected to VIA, everything can be configured in VIA except for the number of layers and the layer names on the display.
All you need to do is take the [VIA.json](https://github.com/Matyas604/Matyas-s-Hackpad/blob/main/Firmware/VIA.json) file and upload it to the VIA website, then connect Hackpad, and you’re ready to configure everything.
